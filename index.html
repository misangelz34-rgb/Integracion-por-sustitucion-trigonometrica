Sustitución trigonométrica — Concepto

Objetivo: transformar radicales cuadráticos en expresiones trigonométricas más simples.

La sustitución trigonométrica es útil para resolver integrales que involucran expresiones como $\sqrt{a^2 - x^2}$, $\sqrt{a^2 + x^2}$ y $\sqrt{x^2 - a^2}$. Usamos identidades trigonométricas para simplificar la raíz y obtener integrales de funciones trigonométricas conocidas.

Casos principales

  • $\sqrt{a^2 - x^2} \Rightarrow x = a\sin\theta$
  • $\sqrt{a^2 + x^2} \Rightarrow x = a\tan\theta$
  • $\sqrt{x^2 - a^2} \Rightarrow x = a\sec\theta$

Con estas sustituciones, el diferencial $dx$ también cambia y la integral puede reescribirse en términos de $\theta$.

Ejemplo 1 — $\int \sqrt{a^2 - x^2}\,dx$

Usamos $x = a\sin\theta$, por lo tanto $dx = a\cos\theta\,d\theta$ y:

$\sqrt{a^2 - x^2} = a\cos\theta$

Entonces:

$\displaystyle \int \sqrt{a^2 - x^2}\,dx = \int a^2\cos^2\theta\,d\theta$

Aplicando la identidad $\cos^2\theta = \tfrac{1}{2}(1+\cos2\theta)$:

$= \tfrac{a^2}{2}\int (1+\cos2\theta)\,d\theta = \tfrac{a^2}{2}(\theta + \tfrac{1}{2}\sin2\theta) + C$

Finalmente, se regresa a $x$ usando $\theta = \arcsin(x/a)$.

Ejemplo 2 — $\int \sqrt{a^2 + x^2}\,dx$

Usamos $x = a\tan\theta$, por lo tanto $dx = a\sec^2\theta\,d\theta$ y:

$\sqrt{a^2 + x^2} = a\sec\theta$

$\displaystyle \int \sqrt{a^2 + x^2}\,dx = \int a^2\sec^3\theta\,d\theta$

Recordando que $\int \sec^3\theta\,d\theta = \tfrac{1}{2}(\sec\theta\tan\theta + \ln|\sec\theta + \tan\theta|)$, se obtiene:

$= \tfrac{a^2}{2}\big(\sec\theta\tan\theta + \ln|\sec\theta + \tan\theta|\big) + C$

Luego se regresa a $x$ usando $\tan\theta = x/a$.

Ejemplo 3 — $\int \sqrt{x^2 - a^2}\,dx$

Usamos $x = a\sec\theta$, por lo tanto $dx = a\sec\theta\tan\theta\,d\theta$ y:

$\sqrt{x^2 - a^2} = a\tan\theta$

$\displaystyle \int \sqrt{x^2 - a^2}\,dx = a^2 \int \tan^2\theta\sec\theta\,d\theta$

Usamos $\tan^2\theta = \sec^2\theta - 1$ para integrar.

Ejercicios propuestos

  1. $\int \frac{dx}{\sqrt{9 - x^2}}$
  2. $\int \frac{x^2}{\sqrt{4 + x^2}}\,dx$
  3. $\int \frac{\sqrt{x^2 - 16}}{x}\,dx$

Sugerencia: identifica el tipo de radical y elige la sustitución trigonométrica adecuada.
